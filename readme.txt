Консольное приложение deliveryConsoleApp

Для указания пути файлам логирования и результата в cmd.exe необходимо указать
первым путь файла логирования, а следующим файл для результата.

deliveryConsoleApp.exe "C:\myLog.txt" "C:\myResult.txt"

Стандартные пути файлов
"log.txt" 
"result.txt"

Файл с заказами хранится в виде JSON файла => order.json
